# ⚡ Atividade: Página Pessoal com Tailwind CSS

Este projeto foi desenvolvido como parte de uma atividade técnica de Desenvolvimento Web, com foco em arquitetura de interfaces modernas, metodologia Mobile-First e publicação em ambiente de produção.

## 🎯 Objetivo da Atividade
Desenvolver uma página pessoal completa e publicá-la, seguindo critérios rigorosos de construção estrutural e restrições tecnológicas específicas.

## 🛠️ Requisitos Atendidos

1. **Metodologia Mobile-First:** O layout foi construído priorizando dispositivos móveis. Através das classes utilitárias do Tailwind (`sm:`, `md:`, `lg:`), a interface se adapta de forma fluida para tablets e desktops, garantindo acessibilidade em qualquer tela.
2. **Estrutura de Conteúdo:**
   - **Página Inicial (Home):** Apresentação profissional e links para redes sociais.
   - **Sobre Mim (Subpágina):** Descrição técnica detalhada e grid de competências (badges).
   - **Portfólio (Subpágina):** Seção com projetos reais e repositórios.
   - **Contato:** Página com formulário estruturado contendo 6 campos (Nome, E-mail, Telefone, Assunto, Mensagem e Checkbox).
3. **Restrição Tecnológica (Sem JavaScript):** Conforme solicitado, o projeto utiliza **estritamente HTML e CSS**. Toda a interatividade, menus responsivos e efeitos de *hover* foram implementados sem a execução de scripts, cumprindo o requisito de "não utilizar JavaScript".
4. **Framework CSS:** Utilização do **Tailwind CSS** para garantir produtividade, design contemporâneo com efeitos de *Glassmorphism* e consistência visual.

## 🚀 Publicação e Entrega
- **Deploy:** Projeto publicado via Vercel/GitHub Pages.
- **Código Fonte:** Arquivo compactado enviado com toda a estrutura necessária para execução local.
